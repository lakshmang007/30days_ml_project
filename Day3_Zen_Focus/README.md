# Zen Focus 🧘

A minimalist, high-aesthetic productivity dashboard designed for deep work.

## ✨ Features
- **Glassmorphic UI**: Modern, sleek design with real-time blur and transparency.
- **Pomodoro Timer**: Stay focused with a 25-minute interval timer.
- **Mindful Task List**: Simple, low-friction task tracking.
- **Zen Aesthetics**: Calm, high-quality backgrounds for a focused environment.

## 🚀 Getting Started
Just open `index.html` in your browser. No installation or build steps required.

## 🛠️ Built With
- HTML5 (Semantic)
- CSS3 (Custom Design System, Glassmorphism)
- Vanilla JavaScript
- Google Fonts (Inter)
